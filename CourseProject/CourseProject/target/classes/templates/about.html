О приложении

Центр обработки документов

Веб-приложение для управления клиентами и документами в центре обработки документов.

Основные возможности:
  • Управление клиентской базой
  • Отслеживание статусов документов
  • Расчет стоимости услуг
  • Управление дополнительными услугами (SMS, доставка)
  • Ролевая система доступа
Роли пользователей:
ADMIN

Полный доступ ко всем функциям, управление пользователями

SUPERVISOR

Просмотр и редактирование всех клиентов

USER

Создание новых клиентов, просмотр только своих записей

READ_ONLY

Только просмотр информации о клиентах

Информация о разработчике:

Приложение разработано в учебных целях с использованием Spring Boot, Spring Security, Thymeleaf и H2 Database.
